**[MeaninglessGPT](https://github.com/tarush-ai/MeaninglessGPT)**: a single-head attention block in NumPy, built to overfit a pattern. The smallest useful transformer I could write.

**[Compiler](https://github.com/tarush-ai/Compiler)**: a PASCAL → MIPS assembly compiler, written in Java.
